## Go (Golang)

### Official Documentation
- **Go.dev** - https://go.dev/doc/
  - Official tutorials and documentation
  - Language specification
  - Standard library reference

### Books

1. **"Let's Go" by Alex Edwards** ⭐ Highly Recommended
   - https://lets-go.alexedwards.net/
   - Build a real web application
   - Best practices from the start
   - Covers Gin-like patterns

2. **"Let's Go Further" by Alex Edwards**
   - https://lets-go-further.alexedwards.net/
   - REST APIs in depth
   - Authentication, rate limiting, CORS
   - JSON handling

3. **"Learning Go" by Jon Bodner**
   - Modern approach to Go
   - Idiomatic patterns
   - Good for beginners

4. **"100 Go Mistakes and How to Avoid Them"**
   - Common pitfalls
   - Best practices
   - Advanced tips

### Online Courses

1. **Go by Example**
   - https://gobyexample.com/
   - FREE hands-on introduction
   - Runnable code examples
   - Perfect quick reference

2. **Tour of Go**
   - https://go.dev/tour/
   - FREE interactive tutorial
   - Official guide
   - Run code in browser

3. **Udemy - "Learn How To Code: Google's Go"**
   - Comprehensive video course
   - Practical exercises
   - Good for visual learners

### Blogs & Articles

- **Three Dots Labs** - https://threedots.tech/
  - Go best practices
  - Clean architecture examples
  - Real-world patterns

- **Go Blog** - https://go.dev/blog/
  - Official Go team posts
  - New features explained
  - Best practices

- **Dave Cheney's Blog** - https://dave.cheney.net/
  - Performance tips
  - Idiomatic Go
  - Deep dives

## Recommended Learning Path

### Month 1: Fundamentals

**Week 1-2:**
- Complete "A Tour of Go"
- Read "Let's Go" (first half)
- Build simple CLI tool in Go

**Week 3-4:**
- Next.js official tutorial
- TypeScript handbook (basics)
- Build simple Next.js app

### Month 2: Integration

**Week 5-6:**
- Build Todo API with Gin
- Connect to PostgreSQL
- Write tests

**Week 7-8:**
- Build Next.js frontend for Todo API
- Practice React Query
- Implement auth

### Month 3: Advanced Topics

**Week 9-10:**
- Clean Architecture patterns
- Refactor Todo app
- Add more complex features

**Week 11-12:**
- Supabase cloud deployment
- Performance optimization
- Production best practices

```bash

go install github.com/golangci/golangci-lint/cmd/golangci-lint@latest

# Database client (optional - use Supabase dashboard)
```

## Final Tips

1. **Learn by Building**
   - Theory is important, but practice more
   - Build real projects
   - Deploy them

2. **Read Code**
   - Explore popular GitHub repositories
   - See how professionals structure code
   - Learn patterns from production code

3. **Join Communities**
   - Ask questions
   - Help others (best way to learn)
   - Stay motivated

4. **Consistent Practice**
   - 1 hour daily > 7 hours once a week
   - Small, consistent progress
   - Track what you learn

5. **Document Your Journey**
   - Blog about what you learn
   - Create notes
   - Helps retention and builds portfolio

Good luck with your learning journey!
